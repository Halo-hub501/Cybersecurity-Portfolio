# Botium Toys Internal Security Audit

## Objective
Conduct a security audit to identify vulnerabilities and improve the organization's security posture.

## Scenario
Botium Toys is expanding internationally and needs to ensure its systems meet security standards and compliance requirements.

## Scope
The audit reviewed the following areas:

- User access controls
- Data protection
- System monitoring
- Network security

## Risks Identified
- Weak password policies
- Lack of multi-factor authentication (MFA)
- Limited system monitoring

## Recommendations
- Implement multi-factor authentication
- Enforce strong password policies
- Deploy centralized logging and monitoring

## Skills Demonstrated
- Security auditing
- Risk assessment
- Compliance analysis
