# Incident Handler's Journal — Template

**Type:** Documentation Tool  
**Used In:** Security incident response and investigation  
**Source:** Google Cybersecurity Certificate — Course 6: Sound the Alarm: Detection and Response

---

## What This Is

The Incident Handler's Journal is a documentation tool used by security analysts to record details about security incidents as they are handled. It captures the key facts of each incident using the **5 W's framework** and serves as a reference log throughout an investigation and after recovery.

This template is part of my active toolkit — I use it in my portfolio project work to document real incident scenarios.

---

## How to Use This Template

1. Fill in the **Date** and **Entry number** for each new incident
2. Write a brief **Description** of the incident
3. List any **Tools** used during the investigation
4. Answer all **5 W's** with 1-2 sentences each
5. Add any **Additional notes**, open questions, or follow-up actions

### The 5 W's

**Who caused the incident?**  
*Identify the threat actor — individual, group, insider, automated malware, etc.*

**What happened?**  
*Describe the attack type, method used, and impact on the organization.*

**When did the incident occur?**  
*Record the date and time the incident was detected or began.*

**Where did the incident happen?**  
*Identify the affected organization, system, network, or location.*

**Why did the incident happen?**  
*Explain the root cause — what vulnerability or weakness was exploited and what the attacker's motivation was.*

### Additional Notes  
*Record open questions, follow-up actions, lessons learned, or anything else relevant.*

## Why This Tool Matters

During a security incident, time is critical and details get lost fast. A structured journal ensures:
- Key facts are captured consistently across all incidents
- Investigators have a clear record to reference during and after response
- Post-incident reviews have accurate timelines and decisions documented
- Evidence is preserved in an organized format for reporting or legal purposes
