# Lab: Install Software in a Linux Distribution

**Module:** Tools of the Trade — Linux and SQL
**Type:** Hands-on Linux Lab (Google Skills / Qwiklabs)
**Tasks Completed:** 5/5 ✅

## Summary

Used the APT package manager in a Debian-based Linux environment to install, uninstall, and verify software applications. Demonstrated the full lifecycle of package management — a critical skill for security analysts who need to manage tools and dependencies on Linux systems.

See `lab-report.md` for the full lab walkthrough.

## Skills Demonstrated
- Verifying package manager availability (`apt`)
- Installing applications via `sudo apt install`
- Uninstalling applications via `sudo apt remove`
- Verifying installation status via `apt list --installed`
- Interpreting package manager output and dependency notices
