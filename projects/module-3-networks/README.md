## What I Learned

This module covers the fundamentals of network security — how networks are structured, how attacks happen at the network layer, and how analysts detect, respond to, and prevent network-based threats.

**Key topics covered:**
- Network architecture and protocols (TCP/IP, DNS, HTTP/S, ARP)
- Common network attacks: DoS, DDoS, SYN flood, DNS poisoning, packet sniffing
- Network hardening techniques: firewall rules, port filtering, network segmentation
- OS hardening: patch management, baseline configurations, MFA
- The NIST Cybersecurity Framework applied to a real incident scenario
